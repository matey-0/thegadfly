Resources Used:
* Hosting - GitHub Pages
* Domain Purchase - namecheap
* Font(s) - Mozilla Fira Mono (Regular, Medium, Bold)
* Email subscription - Buttondown (Free tier) 
* Form - Web3Forms (Free tier)
